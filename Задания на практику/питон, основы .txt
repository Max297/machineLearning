Задача 1: Анализ оценок

Условие:
У вас есть список оценок студентов: [5, 3, 4, 2, 5, 4, 3, 5, 4, 2, 3, 4, 5, 3, 4].
Напишите программу, которая:

    Вычисляет средний балл успеваемости.

    Находит количество неудовлетворительных оценок (оценки ниже 3).

    Определяет, есть ли в списке идеальные оценки (5). Если да, выводит их количество.
	
	
Задача 2: Поиск уникальных элементов 

Условие:
Дан список: [1, 2, 2, 3, 4, 4, 5, 1, 6, 7, 7, 8].
Напишите программу, которая создает новый список, содержащий только уникальные элементы из исходного списка 
(то есть каждый элемент должен встречаться в новом списке только один раз). 
Отсортируйте полученный список по возрастанию

Задача 3: Простой калькулятор

Условие:
Напишите программу-калькулятор, который:

    Запрашивает у пользователя два числа.

    Запрашивает операцию (+, -, *, /).

    Выполняет указанную операцию над числами и выводит результат.

    Если введена неизвестная операция, программа выводит сообщение об ошибке.
	
Задача 4: Анализатор текста

Условие:
Напишите программу, которая анализирует введенный пользователем текст и выводит:

    Количество слов в тексте

    Количество уникальных слов (регистр не учитывать)

    Самое длинное слово в тексте

    Частотный словарь букв (какая буква сколько раз встречается, регистр не учитывать)

    Среднюю длину слова

Задача 5: Игра "Угадай число" с историей и статистикой

Условие:
Разработайте версию игры "Угадай число", где:

    Компьютер загадывает число от 1 до 100

    У игрока есть 10 попыток

    После каждой попытки показывается: "больше" или "меньше"

    Сохраняется история всех ходов игрока

    В конце игры выводится:

        Победил ли игрок

        За сколько попыток

        История ходов

        Статистика (минимальное/максимальное названное число)
